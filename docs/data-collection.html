7 Data collection

As we earlier mentioned, we use data on Amazon customer reviews of beauty products. The data used in this project can be accessed in this link. It contains the following features:

  • Product price
  • Product Id
  • Product title
  • Review helpfulness
  • Review profile Name
  • Review score
  • Review summary
  • Review text
  • Review time
  • Review userId
